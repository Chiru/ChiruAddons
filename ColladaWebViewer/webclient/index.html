    Controls

    Controls on PC

    Rotate: First mouse button

    Pan: Right mouse button

    Zoom: Mouse scroller

    Controls on tablet devices

    Rotate: Move one finger

    Pan: Move two finger on the screen

    Zoom: Use a pinch zoom gesture with two fingers

    Menus

    Right Menu

    Captured objects are shown in the right menu (after connecting to asset server through a WebSocket). You can load a model by clicking the file name in the right menu.

    Models that have been loaded in web browser memory have their filename background colored in green and you can view them instantly.

    Left Menu

    When you have loaded a model from the asset server you can control its scale etc. at the left-side menu.
